## 1. Skip Criteria — Recalibrated

### Hard Skips (Unchanged)
- Healthcare, financial services, or insurance domain **required** (not preferred)
- IC software engineering or hands-on coding required
- SaaS GTM/RevOps specialist required (ARR, NRR, CAC/LTV)
- Contract below $160K annualized or below $75/hr
- Role location more than 20 miles south of Seattle (in-office roles)

| Rule | Old | New |

## 5. ATS Format Rules — Confirm These Are Active

- Font: **Calibri** — not Arial
- Single column layout — no tables anywhere
- Competencies: 3 rows, 5 pipe-separated items per row, plain paragraph text
- No colored text — black only
- No decorative borders or graphic elements
- Contact header: plain text, no text boxes
- Company names: plain text runs — not inside styled or combined runs
- Section headers: bold with simple bottom border only
- No unicode bullet characters — use LevelFormat.BULLET with numbering config in docx builds
